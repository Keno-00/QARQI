# Welcome to QARQI

QARQI (Quadrant Amplitude Representation of Quantum Images) is a framework for representing and simulating images using high-dimensional quantum systems (qudits).

## Overview

Traditional quantum image representations often rely on qubits (2-level systems). QARQI explores the use of d-level systems to improve encoding density and potentially computational efficiency for imaging tasks.

## Main Components

- **QARQICircuit**: Manages the construction and simulation of the image encoding circuits.
- **QARQIResult**: Handles data from simulations, providing easy access to reconstructed images and metrics.
